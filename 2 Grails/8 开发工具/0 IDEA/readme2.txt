. IDEA 如果要开发Grails 应用， 最好使用Ul...版本， 这样， 创建项目的时候就有Grails 选项。

. 选中Grails 后， 右侧会有许多插件选项， 如果一个不选的话， 其实就是一个空项目， 创建后Project Structure 中是空的。 

  运行   Ctrl + Shift + F10

  会有如下提示:

  0 test classes found in package '<default package>'

  Process finished with exit code -1

. 选择 Web Application 选项

  回头再继续试吧， 先在NetBeans 下把功能测试完整了。
